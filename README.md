# Blinkit Sales Analysis Dashboard

## Project Overview

This project is a Power BI dashboard created to analyze Blinkit sales data and generate business insights using interactive visualizations. 
The dashboard helps track key performance indicators (KPIs), sales trends, customer behavior, and product performance.
The project demonstrates skills in:
Data Analysis
Business Intelligence
Power BI Dashboard Development
Data Visualization
KPI Reporting
Data Cleaning & Transformation


## Tools & Technologies Used
Power BI
Power Query
DAX
Data Visualization
Data Modeling

## Key Insights
Total Sales Performance
Product Category Analysis
Customer Purchasing Trends
Outlet/Store Performance
Sales Distribution by Location
KPI Tracking and Business Reporting

## Features
Interactive dashboard with filters and slicers
Dynamic KPI cards
Trend analysis visuals
Category-wise sales breakdown
User-friendly and clean dashboard design
